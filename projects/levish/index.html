levish: Build your own shell

Introduction: What is a shell?

A Shell is a shell.

Installation

levish is on the Python Package Index and can be installed with the following command.


pip install levish

How to get started

Using levish is very easy. We start by importing the 'Shell' class and creating a new instance of it.


from levish import Shell

sh = Shell("MyShell")

Adding functionality

After that we can create our first command. We do that by first creating a simple function. The function has to take varargs called '*args'. Then we can add the function to our Shell object using the add_command() method. Pass in our new created function and add a simple description using the 'description' argument. 


def hello(*args):
    print("Hello there!")

sh.add_command(hello, description="This command greets you.")
